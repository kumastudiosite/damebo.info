# Arrowstream

ブラウザのみで動作するパズルゲームと、ステージの自動生成や検証が可能な開発者ツール（Dev Tool）です。
ビルドツールや外部ライブラリなどの依存はなく、HTML / CSS / JavaScript のみで構成されています。

## 遊び方（Play モード）

1. ブラウザで `src/index.html` を開くか、ローカルサーバーを起動してアクセスしてください。
2. 盤面には色付きの「矢印」が配置されています。
3. 矢印をクリック（タップ）すると、矢尾（Tail）から矢頭（Head）へなぞるように、直進方向にヘビのようについて移動します。
4. **クリア条件**: 交差することなく盤面（画面外）へすべての矢印を完全に退出させるとクリア（LEVEL CLEARED!）です。
5. **交差・衝突の禁止**:
   - 移動中の矢印が、他の矢印の線分（固定されているもの全体）や、自分自身の線分と交差・接触した場合、**即座に停止し**、タップ前の状態までアニメーション付きで巻き戻ります。
   - ペナルティとして、左上の「Cross Events」イベントカウントが `1` 加算されます。
6. **ゲームオーバー**: Cross Events が `3` に達すると、その時点でゲームオーバーとなり、画面操作が無効になります。

## 開発者ツール（Dev Tool モード）

ヘッダー右上の「Dev Tool」ラジオボタンを選択すると、各ステージのレベル生成・検証、およびソルバーの作動確認を行える開発者画面に切り替わります。

### 機能とコントロール

- **W / H**: 盤面のサイズ（幅・高さ）
- **Arrows**: 配置する矢印の目標数
- **Length**: 各矢印の長さの範囲（Min - Max）
- **Turns**: 矢印が曲がる回数の範囲（Min - Max）
- **Seed**: 乱数シード（任意の整数を入力。空欄の場合はランダム生成）
- **Gen N**: 一度に生成試行する盤面の候補数
- **Solvable Only**: チェックを入れると、クリア可能な（探索により minMoves [最小手順] が見つかった）盤面のみをリストに残します。

### 生成とプレイテストのフロー

1. お好みのパラメータを設定し、「**Generate**」をクリックします。
2. ジェネレーターが自動で作動し、条件（初期交差なし、重なり禁止、向かい合い禁止等）を満たす盤面が検索されます。結果のリストが表示されます。
   - ※ W×Hサイズや密度の制約により目標 Arrows 数に達しなかった生成ケースは自動破棄されます。
3. リストから項目をクリックすると、右側のキャンバスにプレビューが表示されます。
   - `Solvable` 判定の場合、必要な最小手数（Moves）も表示されます。
4. リスト項目を選択した状態で「**Playtest Selected**」をクリックすると、そのレベルを保持したまま自動的に Play モード へ遷移し、実際に遊んでテストすることができます。
5. リスト項目を選択した状態で「**Download JSON**」をクリックすると、そのレベルデータを JSON 形式（`arrows`, `grid`, `meta.solution` 等を含む）で保存します。
